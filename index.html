Thrill is a test runner for teams wanting to test their web application on many browsers. Here are some features that you may find... exciting:

  • Shared browser pool for anyone in the network to run tests on.
  • Targeted testing on browsers based on user-agent and capabilities (anything Modernizr can detect).
  • Automatic adaptation of QUnit, jasmine, mocha, or YUI tests, allowing you to run them without any additional configuration.
  • Automatic population of the central server using a Selenium Grid, or from the cloud via Sauce Labs or Browser Stack.
  • Automatic recovery of unresponsive browsers.
  • Run test via command-line, grunt task, or within your own Node.js application.
  • Simple API to write your own adapters or reporters.
  • Use mocha's reporters regardless of the framework you're using.
  • Questionable stability. The most exciting feature of all. Thrill is new, cut it some slack.

Thrill uses Queen, a platform for running scripts on many browsers.
