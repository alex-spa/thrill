Thrill is a test runner for teams needing to test their web application on many browsers.

Browsers connect to a central server. Developers requests tests to be run on the connected browsers remotely. QUnit, mocha, jasmine, and YUI tests work out of the box. Connect testing browsers manually, or automatically using Selenium, Browser Stack or Sauce Labs.

This is free software created by Ozan at Turn.
